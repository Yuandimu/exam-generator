# 智能出题系统 - 服务器部署指南

## 一、环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows / Linux / macOS |
| Python | 3.8 或更高版本 |
| 网络 | 公司内部局域网 |
| 端口 | 8080（可在 `start_server_deploy.py` 中修改） |

## 二、部署步骤

### 1. 安装 Python（如已安装可跳过）

- 下载地址：https://www.python.org/downloads/
- 安装时务必勾选 **"Add Python to PATH"**

### 2. 安装依赖包（首次部署）

打开命令行，进入部署目录，执行：

```bash
pip install python-pptx PyPDF2 python-docx -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 启动服务

**Windows**：双击 `部署启动.bat`

**Linux / macOS**：
```bash
cd 部署目录
python3 start_server_deploy.py
```

### 4. 验证访问

启动后控制台会显示访问地址，例如：

```
本机访问: http://localhost:8080
局域网访问: http://192.168.x.x:8080
```

- 在服务器本机浏览器打开 `http://localhost:8080` 验证
- 在同局域网的其他电脑浏览器打开 `http://服务器IP:8080` 验证

## 三、用户使用流程

1. 浏览器打开系统地址
2. 进入「API配置」页面，填写 DeepSeek API Key（或其他兼容 API）
3. 进入「出题参数」页面，上传课件文件（PPT/PDF/Word/TXT）
4. 设置题量、题型比例、难度比例，点击「生成题目」
5. 在「题库管理」中查看、编辑、筛选题目
6. 进入「组卷中心」选择需要的题目
7. 点击「导出试卷」生成 Word 文档

## 四、配置说明

### 修改端口

编辑 `start_server_deploy.py`，修改第 14 行：

```python
PORT = 8080  # 改为你需要的端口
```

### 修改监听地址

默认监听 `0.0.0.0`（所有网络接口），如需限制为仅本机访问，改为 `127.0.0.1`：

```python
HOST = '0.0.0.0'  # 改为 '127.0.0.1' 则仅本机可访问
```

## 五、后台运行（Linux）

```bash
# 使用 nohup 后台运行
nohup python3 start_server_deploy.py > server.log 2>&1 &

# 查看日志
tail -f server.log

# 停止服务
kill $(lsof -t -i:8080)
```

## 六、常见问题

| 问题 | 解决方案 |
|------|---------|
| 同事无法访问 | 检查服务器防火墙是否放行 8080 端口 |
| 上传文件解析失败 | 确认已安装依赖包（python-pptx, PyPDF2, python-docx） |
| 纯图片 PDF/PPT 无法提取 | 系统仅提取文字，不支持 OCR 图片识别 |
| 端口被占用 | 修改 `start_server_deploy.py` 中的 PORT 值 |
| API 出题失败 | 检查 API Key 是否正确、API 额度是否充足 |
