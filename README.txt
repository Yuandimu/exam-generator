智能出题系统 - 使用说明
=========================

📌 系统简介
-----------
根据上传的课件/文档（PPT/PDF/Word/TXT），自动生成：
- 单选题、多选题、判断题、情境多选题
- 支持 AI 参考出题 + 人工调整
- 一键导出 Word 试卷（.docx）

📋 首次使用（只需做一次）
-------------------------
1. 确保电脑已安装 Python 3.8 或更高版本
   （没装的话去 https://www.python.org/downloads/ 下载，安装时勾选 "Add Python to PATH"）

2. 双击运行 install_deps.bat，自动安装依赖包

🚀 日常使用
-----------
双击 launch.bat 即可启动，浏览器会自动打开系统页面。

🎯 操作流程
-----------
1. 上传文件：点击左侧 "上传文件" 按钮，选择课件/文档
2. 等待解析：系统自动提取文件中的文字内容
3. AI 出题：系统根据内容生成题目建议
4. 调整确认：可修改题目内容、选项、答案
5. 导出试卷：点击 "导出Word" 生成 .docx 文件

📁 支持的文件格式
-----------------
- PPTX  PowerPoint 课件
- PDF   PDF 文档
- DOCX  Word 文档
- TXT   纯文本
- MD    Markdown

❓ 常见问题
-----------
Q: 双击 launch.bat 没反应？
A: 先运行 install_deps.bat 安装依赖，确保 Python 已安装。

Q: 浏览器没有自动打开？
A: 手动打开浏览器访问 http://localhost:3399

Q: 上传文件后提示解析失败？
A: 检查文件是否损坏、是否是纯图片型文件。纯图片 PDF/PPT 无法提取文字。
